# furing_michael
Incident Atlas - is an offline mobile incident location tracker.


Goals

  1. To easily report incident and its exact location to other user.
  2. To easily recieved and track the location of the reported incident without using internet connection.
  
App Feature

  1. Create and send incident location to other user.
  2. Received and display location of the incident.
  3. Send a response message to the user who reported the incident.
  4. Track the location of the incident using the built in map.
  5. List and show incident history.
